To run code, download all files to the same folder, DIR. Run all commands from DIR. Ensure util.py is in the same enclosing folder as DIR. I.e. Folder ML4T contains both DIR and util.py


To run TheoreticallyOptimalStrategy, generate its graphs, and print out its stats, run:
PYTHONPATH=../:. python TheoreticallyOptimalStrategy.py



To run ManualStrategy, generate its graphs, and print out its stats, run: 
PYTHONPATH=../:. python ManualStrategy.py


To generate the indicators graphs alone for in-sample data, run: 
PYTHONPATH=../:. python indicators.py
